# Basit XOX (Tic-Tac-Toe)

Küçük ve modern bir arayüze sahip XOX oyunu. Sadece bir tarayıcı ile çalışır.

## Çalıştırma

- Dosyaları indirin/klonlayın ve `index.html` dosyasını tarayıcıda açın.
- İsterseniz basit bir sunucu ile de açabilirsiniz:

```bash
python3 -m http.server 8080
# ardından tarayıcıdan: http://localhost:8080
```

## Özellikler

- 3x3 tahta, X/O dönüşümlü oynama
- Kazanan/beraberlik tespiti
- Sıfırlama butonu
- Mobil uyumlu basit tasarım

## Dosya Yapısı

- `index.html` – Sayfa iskeleti
- `style.css` – Basit stiller
- `script.js` – Oyun mantığı
